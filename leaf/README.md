# houdini-leaf

[Demo](https://codepen.io/zhua/pen/LYRxxJK)

![Three boxes with leaf and vine in the background](demo.JPG)

Import the script
```html
<script src="https://unpkg.com/houdini-leaf"></script>
```

Default settings
```css
.my-element {
    --leaf-color: black;
    --leaf-size: 16;
    --leaf-variance: left | around;
    background-image: paint(leaf);
}
```
